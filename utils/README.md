# Utilitários de Extração – MetaTrader 5

Este diretório contém scripts auxiliares utilizados no fluxo do robô,
principalmente para extração e tratamento de dados operacionais.

Ferramentas incluídas:

- `extrair_relatorios_mt5.py`
  Script responsável por exportar históricos do MetaTrader 5 para CSV,
  permitindo análise posterior de operações, evolução da estratégia e validação de resultados.

Os arquivos gerados são utilizados no módulo de analytics e revisão de performance.
