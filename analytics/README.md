# Módulo de Analytics – Painel Streamlit

Este diretório contém o painel de análise desenvolvido em Python + Streamlit,
utilizado para visualizar e estudar o desempenho operacional do robô.

Ferramentas incluídas:

- `streamlit_mt5.py`
  Painel interativo para leitura dos arquivos CSV gerados pelo robô e pelo extrator MT5.

Principais objetivos do painel:

- acompanhamento de operações
- análise de lucro x perda
- avaliação de risco
- estudo de comportamento do trailing stop
- apoio à tomada de decisão e ajustes da estratégia
