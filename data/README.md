# Conjunto de Dados – Exportados do MetaTrader 5

Este diretório armazena arquivos CSV gerados a partir:

- do script de extração do MT5
- do registro de execuções do robô
- do módulo de analytics

Os dados incluem:

- `historico_trades_mt5_deals*.csv`
- `historico_trades_mt5_orders*.csv`

Observação:

Os arquivos aqui presentes foram exportados a partir de contas de teste (DEMO)
com o objetivo de validar a automação, registrar comportamento real do robô
e permitir auditoria técnica do funcionamento da estratégia.
